## Protocols

### Changing the base URL of the webpage

1. Goto `src/config/pages.config.tsx`
2. Edit the `WEBSITE_BASE` constant
3. Goto `vite.config.tx`
4. Edit the `base` property
5. Goto `package.json`
6. Edit the `homepage` property
